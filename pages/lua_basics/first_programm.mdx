# Erstes Programm
```lua
print('Hello World')
```
Dies ist wohl das einfachste "Programm". Das einzige was dieser Code macht ist es `Hello World` in der Konsole auszugeben. Hier sieht man auch schon direkt wie man eine Funktion mit einem Parameter auslößt.
Diese Seite soll jedoch erst nur als Beispiel gehen. Am Anfang gehen wir erstmal Variablen durch.
