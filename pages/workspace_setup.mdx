# Workspace einrichten
Grundsätzlich sollte dir klar sein, dass du alles nutzen kannst was du willst. Dies sind aber die Empfehlungen die ich dir geben kann und mit denen ich selber arbeite.

## Code Editor & Extensions
- [VSCode](https://code.visualstudio.com/) - Code Editor
- [LuaLS](https://marketplace.visualstudio.com/items?itemName=sumneko.lua) - Lua Language Server
- [CFXLua](https://marketplace.visualstudio.com/items?itemName=overextended.cfxlua-vscode) - CFXLua Support für den LuaLS
- ([GitLens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) - Besserer Überblick für Code Commits)
- ([GitHub Theme](https://marketplace.visualstudio.com/items?itemName=GitHub.github-vscode-theme) - Theme, rein visuell)
- ([VSCode Icons](https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons) - Icons, rein visuell)

LuaLS und CFXLua sind die wohl wichtigsten Erweiterungen. Diese ermöglichen dir einen Typesupport. Das heißt, dass wenn du zum Beispiel eine Funktion auslöst, die eine Nummer als Parameter haben möchtest, und du einen String übergibst, gibt die Extension dir eine Warnung. Desweiteren kriegst du auch für kleine Fehler wie das Fehlen eines Kommas oder von Klammern Warnungen.
